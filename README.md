# valleyLib

`valleyLib` is a lightweight command-based robotics framework with FTC integration.

## Modules

- `valleyLib-core`: scheduler, commands, command groups, and subsystem abstraction.
- `valleyLib-ftc`: FTC-specific runtime integrations (`OpMode`, telemetry, FTC helper commands).
- `app`: Android host app module.
